# Terra Spotlight

A dynamic web application designed to empower communities in *the Philippines* with timely, actionable information regarding common natural hazards. Our focus is on combining real-time environmental data with clear, structured safety guides to promote preparedness and resilience.

The application serves as a comprehensive dashboard that moves the user seamlessly from immediate situational awareness to detailed safety instructions.

## Key Pages

### Structural & Content Features
- Weather (Right Now): Provides immediate, localized environmental metrics (temperature, wind speed, humidity, and pressure).

- Integrates a dynamic Windy.com map (iframe) for a visual, global context of current wind and precipitation patterns.

### The Hazard Center: The main dashboard for threat assessment.

- Features highly condensed, actionable cards for the common disasters in the Philippines.

- Content is designed for rapid decision-making (e.g., "Brace for Impact!").

### Detailed Safety Guides: Provides in-depth, structured preparedness information.

- Utilizes a Reusable Content Structure where all guides share the same logical layout.

- Information is organized into clear "Before, During, and After" action grids to ensure user predictability.
